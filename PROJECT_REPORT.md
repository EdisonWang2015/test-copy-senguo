### 🚀 运行步骤

**第一步：安装依赖**
```bash
pip install -r requirements.txt
```

**第二步：启动后端（终端1）**
```bash
chmod +x start.sh
./start.sh
```

**第三步：启动前端（终端2）**
```bash
chmod +x start_frontend.sh
./start_frontend.sh
```

**第四步：访问应用**
在浏览器中打开：
```
http://127.0.0.1:8000/frontend/index.html
```

**第五步：运行测试（可选，终端3）**
```bash
# 后端单元测试
cd backend && pytest test_api.py -v

# UI自动化测试（需要后端和前端都在运行）
cd frontend && pytest test_ui.py -v -s
```

### 📞 常见问题处理

1. **后端无法启动？**
   - 检查Python版本（需要3.8+）
   - 确认虚拟环境已激活
   - 查看是否有端口占用

2. **前端无法访问？**
   - 确认前端服务在运行
   - 检查浏览器访问URL是否正确
   - 清除浏览器缓存

3. **测试失败？**
   - 确保依赖已全部安装
   - 检查后端服务是否运行
   - 查看测试输出的错误信息
