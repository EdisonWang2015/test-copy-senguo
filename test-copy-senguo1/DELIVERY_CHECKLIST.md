1. `POST /api/purchase/create` - 创建采购单

4. `PUT /api/purchase/<id>` - 更新采购单

## 🚀 部署说明

### 最小化部署

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动后端 (终端1)
./start.sh

# 3. 启动前端 (终端2)
./start_frontend.sh

# 4. 访问应用
# 打开浏览器: http://127.0.0.1:8000/frontend/index.html
```

### 运行测试

```bash
# 后端测试
cd backend && pytest test_api.py -v

# 前端测试 (需要后端和前端都运行)
cd frontend && pytest test_ui.py -v -s
```
